SemiLagrangian.UniformMesh — Type
UniformMesh{T}
UniformMesh(start::T, stop::T, length::Int) where {T}

1D uniform mesh data.

Arguments

  • start::T : beginning of the mesh
  • stop::T : end of the mesh
  • length::Int : number of cells of the mesh

Implementation

  • step::T : size step
  • points::Vector{T}: Array with node positions
  • width::T : Distance between left and right edges.

SemiLagrangian.interpolate! — Function
interpolate!( fp::AbstractVector{T}, 
    fi::AbstractVector{T},
    decint::Int, 
    precal::Vector{T}, 
    interp::AbstractInterpolation{T, CircEdge, order},
    tabmod=gettabmod(length(fi)) ) where {T, order}

apply an offset to the function fi interpolate by interp struct, the result is in fp vector, decint and precal are precompute with get_precal method, the TypeEdge is CircEdge

Arguments

  • fp::AbstractVector : output vector
  • fi::AbstractVector : input vector
  • decint : offset in units of dx
  • precal::Vector : vector of length order+1 precompute with get_precal(interp, dec) (dec is the offset)
  • interp::AbstractInterpolation{T, CircEdge, order} : interpolation implementation, note that TypeEdge is CircEdge
  • tabmod=gettabmod(length(fi)) : precompute for "begin at one" modulo

Returns :

  • No return
